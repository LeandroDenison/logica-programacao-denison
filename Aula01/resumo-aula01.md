# Aula 01 — Introdução à Lógica de Programação

## O que é um algoritmo
Um algorítmo é uma sequência de ações lógicas que servem para chegar a um resultado.

## Entrada, processamento e saída

Entrada: Essa parte seria quando o usuário insere algo no sistema ex, Nome: Edu, Salário: 1900, Imposto: 250 inss, 100 fgts.
Processamento: Seria como se fosse o calculo necessário para chegar no resultado ex
Nome = Edu
Salário líquido = Salário - Inss - fgts.
Saída: Nome = Edu
       Salário líquido =  1550

## Palavras-chave da aula

Algoritmo — sequência de passos para resolver um problema.  (seria o que eu fiz acima ou também pode ser uma equação matemática por exemplo)
Entrada — dados recebidos pelo sistema.  (são os dados que o sistema irá ter para trabalhar com eles.)
Processamento — transformação dos dados.  (seria o processamendo dos dados inseridos)
Saída — resultado final do processamento.  (Seria a parte em que é mostrado o resultado ao usuário.)
Lógica — forma organizada de pensar para resolver problemas.
